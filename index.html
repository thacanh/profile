About

Selected Projects

LAB-Support – Labor Legal Document Assistance Tool

Legal NLP • LLM • Graph-RAG

A support tool for businesses and individuals to search, create, and manage labor law-related documents in Vietnam.

  • Integrated LLM with Graph-RAG to extract legal clauses.
  • Automatically checks contracts against current regulations.

One Sample, All Counted: A Self-Learning Detection System

Computer Vision • YOLO • ResNet

An object counting pipeline that identifies and counts similar objects from a user-provided example image.

  • YOLO for detection, ResNet for feature extraction.
  • Cosine similarity & semi-supervised refinement.

Vietnamese Visual Question Answering

Multimodal AI • ViT • PhoBERT • BARTpho

A Vietnamese VQA system that answers natural-language questions about given images.

  • ViT for image understanding, PhoBERT for question encoding.
  • BARTpho for free-form answer generation.
