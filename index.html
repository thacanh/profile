Selected Projects

LAB-Support ‚Äì Labor Legal Document Assistance Tool

Developed a tool for businesses and individuals to search, create, and manage labor law-related documents in Vietnam.

  • Integrated a large language model with Graph-RAG to extract legal clauses from documents.
  • Implemented document creation and editing features to reduce legal risks by automatically cross-referencing labor contracts with current regulations.

Keywords: LLMs, Graph-RAG, Legal NLP, Document Processing

One Sample, All Counted: A Self-Learning Detection System

Designed an object counting pipeline to identify and count similar objects from a user-provided example.

  • Used YOLO for object detection and ResNet for feature extraction.
  • Applied cosine similarity for pseudo-label generation and semi-supervised refinement, suitable for industrial and robotic deployment.

Keywords: YOLO, ResNet, Semi-supervised Learning, Object Counting

Vietnamese Visual Question Answering

Built a Vietnamese VQA system combining ViT for image understanding and PhoBERT for question encoding.

  • Replaced traditional classification heads with BARTpho for free-form answer generation.
  • The model accepts an image and a Vietnamese question and returns accurate, natural-language answers.

Keywords: ViT, PhoBERT, BARTpho, VQA, Multimodal AI
